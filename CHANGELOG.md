# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by **Keep a Changelog** and follows **Semantic Versioning**.

---

## [0.5.0]

### Added

- Hardware optimization module.
- Hardware configuration options.
- Hardware service optimization.
- Hardware registry optimization.
- Hardware execution reporting.
- Integration of the Hardware module into the main runner.
- Network optimization module.
- Network configuration options.
- Network optional feature optimization.
- Network service optimization.
- Network registry optimization.
- Network execution reporting.
- Integration of the Network module into the main runner.
- Technical reference documentation for Hardware and Network.

### Improved

- Extended the modular framework with Hardware and Network domains.
- Expanded execution reporting to include Hardware and Network counters.
- Improved module taxonomy by separating physical devices from network protocols.
- Updated configuration documentation for version 0.5.0.

### Changed

- Project version updated to `0.5.0`.
- Network responsibilities clarified separately from Hardware responsibilities.
- IPv6 intentionally excluded from the initial Network module because of compatibility and support considerations.
