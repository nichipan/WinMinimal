# Code of Conduct

WinMinimal is intended to be a respectful and constructive open-source project.

Everyone participating in the project is expected to behave with professionalism and courtesy.

---

# Expected Behavior

Participants are expected to:

- be respectful;
- be constructive;
- assume good faith;
- focus on the technical discussion;
- help make the project welcoming and useful.

---

# Unacceptable Behavior

Unacceptable behavior includes:

- harassment;
- personal attacks;
- intentionally disruptive behavior;
- insulting or demeaning comments;
- repeated off-topic arguments.

---

# Enforcement

The maintainer may moderate discussions, close issues or remove comments that violate this code of conduct.

The goal is not to be bureaucratic, but to keep the project useful, technical and welcoming.

---

# Final Note

Good technical discussion requires disagreement.

Disagreement is welcome when it remains respectful, evidence-based and constructive.
